# ThreeDoors 主线剧情设计文档（方案：终站谢幕）

## 0. 开场与背景

整场游戏发生在一座名为「假面剧场」的巨大舞台空间中。走廊、门扉、怪物与商人，都是演出系统的一部分。每位闯入者都会被卷入一场早已编排却又失控的终幕：**精灵飞贼盗走了终幕剧本（剧中亦称「命运乐章」）**，系统失去既定轨道，所有人被迫在没有完整剧本的情况下继续演下去。

---

## 1. 主轴与核心命题

- **主轴**：整场冒险是一出被迫继续的剧场演出，舞台为“假面剧场”。  
- **起因**：精灵飞贼（银羽飞贼）盗走了最关键的**终幕剧本**，导致既定演出系统失稳。  
- **核心命题**：当“角色失去剧本”后，玩家是否还能通过自己的选择完成一场有意义的谢幕。

---

## 2. 世界观设定（统一解释）

| 事件/元素 | 在主世界观中的真实身份 | 对主线的意义 |
|---|---|---|
| 精灵飞贼（银羽飞贼） | 剧本盗取者 | 制造“无剧本演出”的起点 |
| 黑暗木偶（弃线木偶） | 失去剧本控制的原主角演员 | 展示失控样本与悲剧后果 |
| 月蚀通缉令 | 剧场安保回收系统 | 想追回剧本，但长期抓错人 |
| 移动摊贩/黑市列车摊 | 门厅自动售票系统的故障分流终端 | 提供漏洞票并引出查票清算 |
| 梦境井 | 历代演出的回放池 | 展示不同历史结局与代价 |
| 假面剧场 | 主舞台与终幕场地 | 所有线索最终汇流并完成谢幕 |
| 命运乐章/命运乐谱 | 剧本的剧中称谓 | 月蚀通缉与审判线中“被追回之物”的具体名称 |

## 3. 事件关联与玩家选择（与当前实现对齐）

### 3.1 精灵飞贼（银羽飞贼 · 失窃起点）

- **与主世界观关联**：她偷走了终幕剧本（命运乐章），直接造成“无剧本状态”。
- **起始事件**：银羽飞贼（暗巷初遇）。
- **玩家可选方向**：递上口粮表示合作 / 拔刀试探先打一场 / 谎称守卫逼她交赃。
- **后续链**：银羽暗号 → 屋脊切磋 → 假地图与真考验 → 怪物门前的演武 → 夜营火谈 → 陷阱回廊 → 猎门同调 → 双人盗案（剧本残页存放室）→ 银羽余响（收束）。
- **银羽余响分支**：按关系值（高/低/中）呈现不同收尾；可选接过钥匙结盟、礼貌道别、或拒绝/决裂。
- **对最终走向的影响**：关系高则终章可获得“剧本真页”或飞贼证词；敌对则可能被揭发，提高失败风险。

### 3.2 黑暗木偶（弃线主演）

- **与主世界观关联**：木偶是失去剧本控制的原定主演，崩溃后呈现蓝光善侧与红噪暗侧双重人格。
- **事件链顺序**：弃线木偶（仅规避）→ 失真童谣 → 蓝眼回声 → 人格裂隙 → 核心下潜 → 终战（堕暗机偶·黑暗完全体）。
- **弃线木偶**：钻进井盖潜行 / 拉闸熄灯撤离 / 丢诱饵芯片引开。
- **失真童谣**：重放温和语音样本 / 播放战术日志分析 / 打包转卖设备换物资。
- **蓝眼回声**：相信善侧按路线前进 / 追问被弃过去稳定情绪 / 记录情感弱点准备压制。
- **人格裂隙**：护住善侧信号通道 / 维持双信号平衡 / 向暗侧发送毁灭指令。
- **核心下潜**：写入修复补丁 / 切断情感模块 / 随机录入指令试图控制本体。
- **对最终走向的影响**：善侧选择降低终战难度并影响结局文案（晨光修复/带伤停机等）；暗侧选择强化 Boss 并偏向权力型结局。

### 3.3 月蚀通缉令（错误追缉）

- **与主世界观关联**：剧场安保回收系统，通缉「命运乐谱大盗」、追回「命运乐章」，但其实追错了人，命运乐谱就是剧本，真正偷走的人是精灵飞贼；批注“目标身份待复核”。
- **起始事件**：月蚀通缉令。
- **玩家可选方向**：接单追猎拿下命运乐谱大盗 / 撕毁通缉令并暗中护送目标 / 两边伪造线索等他们互咬后收网。
- **中段**：强制触发一战（接单→与大盗战；护送→与命运乐章守护者战；双吃→随机一方上门），战后可获旧日记本，审判庭要求携证出庭。
- **收束事件**：月蚀审判（按规矩结案 / 销毁证物 / 反向勒索审判庭）；结案后可获得复活卷轴等宝物门奖励，或触发追责。
- **对最终走向的影响**：合法结案则安保背书、终章阻力小；销毁/勒索则后续可能被追缴或反噬。

### 3.4 齿轮售票亭与查票清算（门厅系统）

- **与主世界观关联**：门厅自动售票终端故障，修好可换正规入场券，偷码可白嫖漏票，破坏会树敌。
- **起始事件**：齿轮售票亭。选项：校准售票机关换正规入场券 / 偷看优惠码白嫖漏票 / 破坏摊位抢材料。
- **收束事件**：查票清算。选项：补款结清票务 / 伪造通行证硬闯 / 贿赂计价员买断风声。
- **对最终走向的影响**：校准路线利于合法身份与后续奖励；白嫖/破坏会触发回滚、扣款或清算，影响终章资源与身份。

### 3.5 梦境井与回声法庭（历史回放）

- **与主世界观关联**：梦境井映出历代梦境的三段结局回放；交易会被回声法庭立案。
- **起始事件**：梦境井。选项：喝下井水读取梦境回放 / 封住井口拒绝回放 / 把回放卖掉变现。
- **收束事件**：回声法庭。选项：赎回回放记忆 / 上缴回放罚款 / 继续倒卖回放。
- **对最终走向的影响**：赎回偏向秩序与后续补给；倒卖带来短期收益但可能触发追缴与伤害，影响终章立场。

### 3.6 镜面剧场（预演厅）

- **与主世界观关联**：假面剧场的预演厅，抉择会写进下一段命运。
- **选项**：戴上英雄面具 / 戴上恶徒面具 / 撕掉剧本离场（撕本可能触发回声法庭）。
- **对最终走向的影响**：影响道德值与后续随机事件（补给/伏击/折扣等），为终章立场埋线。

### 3.7 主线事件链一览（实现顺序）

| 长链 | 事件序列（按触发顺序） |
|------|------------------------|
| 银羽飞贼 | 银羽飞贼 → 银羽暗号 → 屋脊切磋 → 假地图与真考验 → 怪物门前演武 → 夜营火谈 → 陷阱回廊 → 猎门同调 → 双人盗案 → 银羽余响 |
| 弃线木偶 | 弃线木偶 → 失真童谣 → 蓝眼回声 → 人格裂隙 → 核心下潜 → 终战（堕暗机偶/黑暗完全体） |
| 月蚀通缉 | 月蚀通缉令 → 中段强制战斗（大盗/守护者/随机）→ 月蚀审判 |
| 售票/查票 | 齿轮售票亭 → 查票清算 |
| 梦境井 | 梦境井 → 回声法庭（喝下路线强制触发） |
| 镜面剧场 | 镜面剧场（独立，撕本可链到回声法庭） |

## 4. 最终结局扩展（增强版）

### 4.0 当前默认结局：选择困难症候群（已实现）

- **触发条件**：回合数达到 200 时，强制进入普通结局门廊（未触发其他长链终战的前提下）。
- **流程**：  
  1. **终局回廊·第一门**：三道门牌写着「快一点」「再想想」「都可以」，玩家选一扇推开。  
  2. **终局回廊·第二门**：三道门仅标「左」「中」「右」，再次选择。  
  3. **终战**：三扇终局门闭合，中央裂隙中登场 Boss「选择困难症候群」；门缝提示：「看提示看了两百回合，终于舍得进来啦？」  
  4. **通关**：击倒后触发【普通结局】——你击倒了「选择困难症候群」，抵达迷宫出口并离开。
- **剧场化解读**：在假面剧场中，若玩家未走入任一长链终幕（银羽/木偶/月蚀等），迷宫会在第 200 回合把「迟迟不做决定」本身具现为最终对手——选择困难症候群既是对“反复看提示、三扇门每次都要想半天”的调侃，也是“在无剧本状态下必须做出决定才能谢幕”的收束。
- **与 §4.1–4.4 的关系**：§4.1–4.4 为设计目标中的多结局方向；当前实现以本默认结局作为通用收尾，后续可与证词/身份/债务等结算整合。

### 4.1 结局A：补全谢幕（秩序结局）

- **定义**：玩家尽可能找回剧本并按结构完成终演。  
- **世界状态**：秩序稳定、风险最低，但角色能动性被压缩。  
- **典型前提**：  
  - 飞贼线偏合作或可回收残页  
  - 通缉线偏合法结案  
  - 售票线偏正规购票/低债务  
  - 梦境井立场偏“稳定优先”

## 4.2 结局B：即兴谢幕（自由结局）

- **定义**：承认剧本不可完整复原，选择以角色意志完成终演。  
- **世界状态**：不确定性最高，但演出不再被旧模板束缚。  
- **典型前提**：  
  - 飞贼线对你信任或提供关键口述真相  
  - 木偶线偏修复善侧人格  
  - 梦境井立场偏“自由优先”  
  - 终章拒绝安保系统对“标准谢幕”的强制定义

## 4.3 结局C：接管谢幕（权力结局）

- **定义**：玩家以“导演代理人”身份接管舞台规则。  
- **世界状态**：短期高效统一，长期取决于玩家统治逻辑。  
- **典型前提**：  
  - 通缉线与售票线都能被你操纵（合法或非法都可）  
  - 木偶线偏压制或工具化  
  - 梦境井立场偏“控制优先”  
  - 终章中选择“规则由我定义”

## 4.4 结局D：破场流产（失败结局，可选）

- **定义**：证词冲突、债务爆发、安保清算同时失控，谢幕未完成。  
- **世界状态**：舞台坍缩，演出中断，进入混乱残响。  
- **典型前提**：  
  - 飞贼、木偶、安保三方关系同时恶化  
  - 售票/查票债务高位爆发  
  - 梦境井立场与前置行为严重矛盾（例如口头求稳但全程极端套利）

## 5. 前置事件如何改变最终结局（影响矩阵）

## 5.1 精灵飞贼线对终局的影响

- **高信任**：终章可获得“剧本真页”或飞贼证词，显著提升补全/即兴成功率。  
- **中立**：只能拿到碎片信息，终章需要更多其他线补证。  
- **敌对**：飞贼可能在终章指认你，触发安保侧压力并提高失败风险。

## 5.2 黑暗木偶线对终局的影响

- **修复善侧**：终章获得“旧主演自证”，可降低舞台暴走风险。  
- **工具化利用**：可换来短期战斗/资源优势，但终章可信度下降。  
- **彻底压制**：有利于接管型结局，但不利于自由型结局的正当性。

## 5.3 月蚀通缉线对终局的影响

- **合法结案**：安保系统为你背书，终章流程阻力小。  
- **灰色操作**：终章可用但风险高，容易被追责反噬。  
- **高强度欺诈**：终章时可能触发额外拦截，压缩结局选择空间。

## 5.4 售票/查票线对终局的影响

- **正规票务**：终章资源稳定，叙事身份正当。  
- **漏洞白嫖**：中后段触发查票，造成部分罚没但仍可挽回。  
- **伪票倒卖**：高概率在终章前爆发大额清算，显著提高失败结局权重。

## 5.5 梦境井立场对终局的影响

- **稳定优先**：补全谢幕权重上升。  
- **自由优先**：即兴谢幕权重上升。  
- **控制优先**：接管谢幕权重上升。  
- **立场反复横跳**：若与前序行为冲突过大，会提高“破场流产”触发率。

## 6. 结论（给后续实现的叙事要求）

- 该版本文档仅保留三类信息：  
  1) 主世界观与统一解释  
  2) 各事件与主世界观的关联  
  3) 玩家选择如何改变最终谢幕走向
- 终章不再是单一 Boss 收尾，而是“证词、身份、债务、立场”的综合结算。  
- 只要前置事件被正确记录，终局就能形成可解释的多结局闭环。
- 本文档 §3、§3.7 已按当前实现的事件名与分支选项整理，便于与代码（如 `events.py`、`story_system.py`）对照；§4 结局分类为设计目标，与现有终战 Boss 及结局门可逐步对齐。

## 7. 非主线事件的剧场化融入

以下事件虽非主线关键，但可在叙事上统一为「剧场走廊中的遭遇」，与主世界观保持协调：

| 游戏内事件 | 剧场化解释 | 说明 |
|---|---|---|
| 受伤陌生人 | 走廊中受伤的群众演员/临时演员 | 演出失控后留下的伤者 |
| 走私犯 | 兜售后台通行证或漏票的票贩 | 与门厅故障售票系统相关 |
| 古老祭坛 | 剧场侧厅的仪式台，刻有旧演出符文 | 旧秩序残留 |
| 流浪赌徒 | 走廊里设局的赌档，赌注是入场券或金币 | 演出间隙的灰色交易 |
| 迷路孩童 | 迷失在舞台布景中的小演员/观众 | 剧场空间错位 |
| 镜剧场 | 假面剧场的预演厅或镜像空间 | 主舞台的试炼分支 |
| 时光当铺 | 剧场的存包处 | 观众/演员寄存物品的终端，「收今天押明天」对应存包与取回 |
| 逃难队伍 | 逃票进入剧场的一批人 | 未购票混入，请求不要声张并赞助路费以继续躲藏 |
| 落难骑士 | 失去剧本指引的旧戏角色 | 重伤倒地、不知如何行动，可选帮助/搜刮/离开 |
| 诅咒宝箱 | 旧时代演出的遗留道具箱 | 「贪婪者必受惩罚」的警告与强行打开/净化/离开 |
| 智者 | 旧剧本里的导师型角色 | 问「为了什么而踏上这舞台」，可选力量/财富/生存 |
