# ThreeDoors 主线剧情设计文档（方案：终站谢幕）

## 1. 主轴与核心命题

- **主轴**：整场冒险是一出被迫继续的剧场演出，舞台为“假面剧场”。  
- **起因**：精灵飞贼盗走了最关键的终幕剧本，导致既定演出系统失稳。  
- **核心命题**：当“角色失去剧本”后，玩家是否还能通过自己的选择完成一场有意义的谢幕。

---

## 2. 世界观设定（统一解释）

| 事件/元素 | 在主世界观中的真实身份 | 对主线的意义 |
|---|---|---|
| 精灵飞贼（银羽飞贼） | 剧本盗取者 | 制造“无剧本演出”的起点 |
| 黑暗木偶（弃线木偶） | 失去剧本控制的原主角演员 | 展示失控样本与悲剧后果 |
| 月蚀通缉令 | 剧场安保回收系统 | 想追回剧本，但长期抓错人 |
| 移动摊贩/黑市列车摊 | 门厅自动售票系统的故障分流终端 | 提供漏洞票并引出查票清算 |
| 梦境井 | 历代演出的回放池 | 展示不同历史结局与代价 |
| 假面剧场 | 主舞台与终幕场地 | 所有线索最终汇流并完成谢幕 |

---

## 3. 事件关联与玩家选择（仅保留主线关键）

## 3.1 精灵飞贼（失窃起点）

- **与主世界观关联**：她偷走了终幕剧本，直接造成“无剧本状态”。
- **玩家可选方向**：  
  - 协助她藏匿残页  
  - 强夺残页并追问真相  
  - 交易情报、暂时放行
- **对最终走向的影响**：  
  - 决定她在终章是“返场协助”“中立旁观”还是“反向揭发你”。  
  - 决定玩家能否在终章拿到“剧本真页”或仅持有碎片版真相。

## 3.2 黑暗木偶（弃线主演）

- **与主世界观关联**：木偶并非随机怪物，而是原定主演在失去剧本后崩溃的结果。
- **玩家可选方向**：  
  - 稳定善侧人格  
  - 利用其失控获取收益  
  - 彻底压制并切断人格
- **对最终走向的影响**：  
  - 决定终章中“旧主演证词”是否可用。  
  - 决定终章舞台冲突是可控（低压）还是暴走（高压）。  
  - 决定玩家是否能把“旧剧本秩序”转化为“新规则秩序”。

## 3.3 月蚀通缉令（错误追缉）

- **与主世界观关联**：这是回收剧本的安保系统，但由于识别模板落后而误捕玩家/木偶。
- **玩家可选方向**：  
  - 配合调查、走正规结案  
  - 伪造证据转移目标  
  - 黑吃黑并利用追缉牟利
- **对最终走向的影响**：  
  - 决定终章时“安保系统”是你的背书者还是拦截者。  
  - 决定你能否获得合法入场权（影响终幕发言权和流程控制权）。  
  - 决定观众侧叙事是“被冤者”还是“高风险嫌犯”。

## 3.4 移动摊贩/售票故障（门厅系统）

- **与主世界观关联**：本质是剧场自动售票系统故障，黑市化后成为流动摊贩剧情。
- **玩家可选方向**：  
  - 修复系统并正规购票  
  - 利用漏洞白嫖免费票  
  - 伪造票据并批量倒卖
- **对最终走向的影响**：  
  - 决定终章前是否触发查票清算。  
  - 决定终章资源状态（补给、罚没、债务）。  
  - 决定终章时你是“合法观演者/共演者”还是“待清算对象”。

## 3.5 梦境井（历史回放）

- **与主世界观关联**：提供历代演出的失败回放，让玩家看到过去为何谢幕失败。
- **玩家可选方向**：  
  - 接受旧秩序，优先稳定  
  - 坚持即兴，优先自由  
  - 以控制为目标，接管演出
- **对最终走向的影响**：  
  - 直接决定终章可选结局集合。  
  - 决定玩家在终章的价值立场（秩序/自由/权力）。  
  - 决定“是否复原剧本”“复原到什么程度”。

## 3.6 假面剧场（主舞台）

- **与主世界观关联**：这是所有支线与证词汇流的最终审议台。  
- **玩家可选方向**：  
  - 用证词拼回旧剧本  
  - 拒绝旧剧本并即兴谢幕  
  - 接管舞台规则并重写演出
- **对最终走向的影响**：  
  - 直接生成最终结局类型。  
  - 决定世界进入“稳定延续 / 不确定自由 / 强控秩序”哪一种后续状态。

---

## 4. 最终结局扩展（增强版）

## 4.1 结局A：补全谢幕（秩序结局）

- **定义**：玩家尽可能找回剧本并按结构完成终演。  
- **世界状态**：秩序稳定、风险最低，但角色能动性被压缩。  
- **典型前提**：  
  - 飞贼线偏合作或可回收残页  
  - 通缉线偏合法结案  
  - 售票线偏正规购票/低债务  
  - 梦境井立场偏“稳定优先”

## 4.2 结局B：即兴谢幕（自由结局）

- **定义**：承认剧本不可完整复原，选择以角色意志完成终演。  
- **世界状态**：不确定性最高，但演出不再被旧模板束缚。  
- **典型前提**：  
  - 飞贼线对你信任或提供关键口述真相  
  - 木偶线偏修复善侧人格  
  - 梦境井立场偏“自由优先”  
  - 终章拒绝安保系统对“标准谢幕”的强制定义

## 4.3 结局C：接管谢幕（权力结局）

- **定义**：玩家以“导演代理人”身份接管舞台规则。  
- **世界状态**：短期高效统一，长期取决于玩家统治逻辑。  
- **典型前提**：  
  - 通缉线与售票线都能被你操纵（合法或非法都可）  
  - 木偶线偏压制或工具化  
  - 梦境井立场偏“控制优先”  
  - 终章中选择“规则由我定义”

## 4.4 结局D：破场流产（失败结局，可选）

- **定义**：证词冲突、债务爆发、安保清算同时失控，谢幕未完成。  
- **世界状态**：舞台坍缩，演出中断，进入混乱残响。  
- **典型前提**：  
  - 飞贼、木偶、安保三方关系同时恶化  
  - 售票/查票债务高位爆发  
  - 梦境井立场与前置行为严重矛盾（例如口头求稳但全程极端套利）

## 5. 前置事件如何改变最终结局（影响矩阵）

## 5.1 精灵飞贼线对终局的影响

- **高信任**：终章可获得“剧本真页”或飞贼证词，显著提升补全/即兴成功率。  
- **中立**：只能拿到碎片信息，终章需要更多其他线补证。  
- **敌对**：飞贼可能在终章指认你，触发安保侧压力并提高失败风险。

## 5.2 黑暗木偶线对终局的影响

- **修复善侧**：终章获得“旧主演自证”，可降低舞台暴走风险。  
- **工具化利用**：可换来短期战斗/资源优势，但终章可信度下降。  
- **彻底压制**：有利于接管型结局，但不利于自由型结局的正当性。

## 5.3 月蚀通缉线对终局的影响

- **合法结案**：安保系统为你背书，终章流程阻力小。  
- **灰色操作**：终章可用但风险高，容易被追责反噬。  
- **高强度欺诈**：终章时可能触发额外拦截，压缩结局选择空间。

## 5.4 售票/查票线对终局的影响

- **正规票务**：终章资源稳定，叙事身份正当。  
- **漏洞白嫖**：中后段触发查票，造成部分罚没但仍可挽回。  
- **伪票倒卖**：高概率在终章前爆发大额清算，显著提高失败结局权重。

## 5.5 梦境井立场对终局的影响

- **稳定优先**：补全谢幕权重上升。  
- **自由优先**：即兴谢幕权重上升。  
- **控制优先**：接管谢幕权重上升。  
- **立场反复横跳**：若与前序行为冲突过大，会提高“破场流产”触发率。

## 6. 结论（给后续实现的叙事要求）

- 该版本文档仅保留三类信息：  
  1) 主世界观与统一解释  
  2) 各事件与主世界观的关联  
  3) 玩家选择如何改变最终谢幕走向
- 终章不再是单一 Boss 收尾，而是“证词、身份、债务、立场”的综合结算。  
- 只要前置事件被正确记录，终局就能形成可解释的多结局闭环。
